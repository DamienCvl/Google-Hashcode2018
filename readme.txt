README - Hashcode 2018 - Partie Arbitre

Etape à suivre :

	1. Un dossier 'Maps' doit être présent à la racine de l'exécutable.
	2. Tout les fichiers de sorties (.out) que vous voulez analyser doivent être dans ce dossier 'Maps'.
	3. L'exécutable écrira les scores dans l'invite de commande.

README - Hashcode 2018 - Partie Solution

Etape à suivre :

	1. Un dossier 'Maps' doit être présent à la racine de l'exécutable.
	2. Tout les fichiers d'entrées (.in) que vous voulez analyser doivent être dans ce dossier 'Maps'.
	3. L'exécutable écrira les fichiers de sorties (.out) et des fichiers visuels (.visu) dans ce dossier 'Maps'.
